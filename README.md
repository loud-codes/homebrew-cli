# homebrew-cli

Homebrew tap for [LOUD](https://loud.codes) — terminal-first AI for builders.

## Install

```bash
brew tap loud-codes/cli
brew install loud
```

## What's in this tap

| Formula | Description |
|---|---|
| `loud` | LOUD CLI — terminal-first AI agent |

## Update

```bash
brew update
brew upgrade loud
```

## Uninstall

```bash
brew uninstall loud
brew untap loud-codes/cli
```
